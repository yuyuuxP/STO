## STO
Smart Task Organizer | Program Pengatur Tugas 

## Deskripsi
Smart Task Organizer (STO) Membantu pengguna dan kelompok untuk mengatur (organize) segala tugas-tugas (task) yang menjadi tanggung jawab pengguna untuk diselesaikan, membantu untuk meringankan pekerjaan pengguna, memperjelas objektifitas tugas yang diberikan kepada pengguna.

Fitur:
* Manajemen akun yang cukup aman menyimpan data pengguna
* Pemberitahuan notifikasi pengingat bagi pengguna untuk menyelesaikan tugasnya
* Mencari tugas berdasarkan nama ataupun menampilkan keseluruhan tugas
* Membuat dan mengatur tugas berdasarkan prioritas pengguna
* Pengguna dapat melakukan perubahan pada tugas yang telah dibuat
* Dapat melakukan restorasi pada tugas yang telah dihapus 
* Pengguna dapat mengatur jadwal deadline tugas dan melihat statistik task yang telah dilakukan

Cocok untuk pengguna perorangan dan kelompok kecil yang membutuhkan alat pengatur tugas sederhana namun fleksibel.

## Keperluan
* Kami menggunakan 2 class yang kami buat yakni User Class sebagai blueprint dari pengguna (user)  dan Task Class sebagai blueprint dari objek tugas (task) yang akan dibuat.
* Kami membutuhkan sebuah tempat penyimpanan data dengan menggunakan metode penyimpanan list dictionary dengan 3 list yakni: 1. akun_list, 2. task_list, 3. trash_list
* Semua dilakukan dengan sangat sederhana, berdasarkan referensi code dari beberapa tugas praktikum terkhusus tugas praktikum 7
